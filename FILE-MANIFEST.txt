================================================================================
                        FILE RELATIONSHIPS
================================================================================

index.html (depends on)
    ├── styles.css (imported for styling)
    ├── script.js (imported for functionality)
    └── data.js (imported for gallery content)

script.js (uses)
    ├── data.js (reads galleryData array)
    ├── DOM elements from index.html
    └── CSS classes from styles.css

styles.css (applies to)
    └── All elements in index.html

data.js (provides content to)
    └── script.js (dynamically rendered)
