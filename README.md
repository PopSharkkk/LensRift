# Studio Portfolio

Personal photography and cinematography portfolio. Left sidebar navigation, masonry grid, image carousel, drag-and-drop uploads.

## Structure

```
/
├── index.html   — single page, all sections
├── style.css    — all styles
├── main.js      — all JavaScript
└── README.md
```

## Features

- Left sidebar navigation with category filtering
- Auto-playing image carousel
- Masonry grid layout with 24 placeholder slots
- Drag-and-drop image upload (stored in localStorage)
- Fullscreen lightbox with keyboard navigation
- Smooth Apple-inspired animations and transitions
- Fully responsive

## Usage

Open `index.html` in a browser, or deploy to Vercel / Netlify / GitHub Pages.

No build step. No dependencies. No frameworks.

## Customization

- **Your name** — replace "Your Name" in `index.html` sidebar
- **Contact email** — update `mailto:` link in sidebar
- **Colors** — edit CSS variables at top of `style.css`
- **Categories** — add/remove `<option>` entries in the upload modal and filter buttons
